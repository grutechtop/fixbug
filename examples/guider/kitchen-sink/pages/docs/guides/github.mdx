# All markdown and components

This page describes all markdown and components that are supported by `@neato/guider`.

## Inline styles
These are all inline styles.

- **This is bold text**
- _This text is italicized_
- ~~This is strikethrough text~~
- **This text is _extremely_ important**
- ***All this text is important***
- This text has `Inline code` inside it.
- The following has inline syntax highlighting:  `const ts = "test"{:ts}`. Very cool!

### Scripts
Sometimes you want to use sub or superscript, this is how that looks:
- This is a <sub>subscript</sub> text
- This is a <sup>superscript</sup> text

## Code blocks

Here is a basic code block:
```
git status
git add
git commit
```

With a language specified:
```sh
git status
git add
git commit
```

Scrollable code block:
```ts
const hello = "world"; const hello = "world"; const hello = "world"; const hello = "world"; const hello = "world"; const hello = "world";
```

Code block with line numbers:
```ts showLineNumbers
const hello = "world";
```

Code block with title:
```ts title="test"
const hello = "world";
```

Code block with npm2yarn integration:
```sh npm2yarn
npm i @neato/guider
```

```ts title="Code block with difference view"
export function foo() {
  console.log('hewwo') // [!code --]
  console.log('hello') // [!code ++]
}
```

```ts title="Code block with difference view and lines" showLineNumbers
export function foo() {
  console.log('hewwo') // [!code --]
  console.log('hello') // [!code ++]
}
```

```ts title="Code block with line highlight"
export function foo() {
  console.log('hello') // [!code highlight]
}
```

```ts title="Code block with word highlight"
export function foo() { // [!code word:hello]
  console.log('hello') // hello world!
}
```

```ts title="Code block with a focus"
export function foo() {
  console.log('Hello') // [!code focus]
}
```

```ts title="Code block with notations"
export function foo() {
  console.error('Error') // [!code error]
  console.warn('Warning') // [!code warning]
}
```

```ts {1,3-4} title="Code block with meta line highlight"
console.log('1')
console.log('2')
console.log('3')
console.log('4')
```

```ts /Hello/ title="Code block with meta word highlight"
export function foo() {
  const msg = 'Hello World'
  console.log('Hello')
}
```
